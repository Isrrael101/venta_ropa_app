INSTRUCCIONES PARA EL ICONO

Para generar el icono de la aplicación, necesitas seguir estos pasos:

1. Crea una imagen PNG de 1024x1024 píxeles que represente tu app
   - Puede ser un logo simple con colores que representen ventas de ropa
   - Recomendamos usar azul (para BOB) y verde (para ARS) en el diseño
   - Incluye el texto "VENTAS" en la imagen

2. Guarda esta imagen como "icon.png" en esta carpeta (assets/icon/)

3. Ejecuta el siguiente comando para generar los iconos:
   flutter pub run flutter_launcher_icons

4. Esto generará automáticamente los iconos para Android, iOS y otras plataformas

RECOMENDACIONES DE DISEÑO:
- Mantén el diseño simple y reconocible
- Usa colores contrastantes
- Asegúrate que sea reconocible incluso en tamaños pequeños
- Evita detalles muy pequeños que no se verían bien en iconos pequeños
